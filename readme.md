# Recommendation Systems

## This repository contains various projects related to recommendation systems, including collaborative filtering, content-based filtering, and hybrid approaches. Each project is designed to demonstrate different techniques and algorithms used in building recommendation systems.

## Projects
- [Movie Recommender](./Movie_Recommender): A movie recommender system that uses an LLM on Content-based recommendatation to tell about the genres, cast, and summarize a brief overview of the movies.

- [Book Recommender](./Book_Recommender): A simple book recommender system that uses content-based filtering to recommend books based on user ratings.
